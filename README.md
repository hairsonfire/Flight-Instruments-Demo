# Flight Instruments Demo

A flight instrument simulator written in C++ and Qt.

## Features

- Artificial Horizon
- Altimeter
- Real-time simulation loop
- Keyboard controls
- Model / View architecture
- Qt signals and slots

## Technologies

- C++20
- Qt 6
- CMake
- QPainter
